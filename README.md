# Генератор печаток · Україна

Веб-генератор круглих, овальних, прямокутних і трикутних печаток за стандартами України. Без серверу — все працює локально в браузері.

**🚀 Демо:** https://dmmat.github.io/stamp/

## Можливості

- Круглі (Ø 25–45 мм), овальні, прямокутні, трикутні форми
- 3 кільця з настроюваним діаметром і товщиною (38 мм / 36 мм / 28 мм за замовчуванням)
- Декоративні візерунки внутрішнього кільця: пунктир, зигзаг, меандр, ланцюг, мотузка, плетіння, кабель, крапки
- Захисні фонові узори (гільош): крапки, діагональ, сітка, хвилі, муар тощо
- Текст по 3 дугах (верхній, нижній, середній «Код …»), окремий шрифт для кожної
- Центральний блок до 6 рядків з індивідуальним шрифтом / жирністю / курсивом / розміром
- Розділювач `*` у дугах рендериться більшим (множник налаштовується)
- Експорт у **SVG**, **PNG (300 dpi)**, **DXF (R12 для AutoCAD/Fusion 360)**, **JSON** (для збереження налаштувань)
- Імпорт JSON — повне відновлення печатки
- Конвертація тексту у криві (через `opentype.js` + open-source шрифти)
- Перетворення `stroke` у заповнені області — щоб у CAD/CNC лінії мали видиму товщину
- Підкладка з фотографією для звірки з реальним відбитком (drag, scale, rotate, opacity)
- Шкала превʼю 50–600%, лінійка 1 мм, паперовий фон

## Запуск

```bash
# 1. Клонуй репо
git clone https://github.com/dmmat/stamp.git
cd stamp

# 2. (Перший раз) завантаж локальні шрифти, потрібні для конвертації у криві
./setup-fonts.sh

# 3. Підніми статичний сервер
python3 -m http.server 8000
# або просто відкрий index.html у браузері
```

## Власні пресети

`presets.json` — публічні шаблони, що йдуть з репо. Якщо хочеш приватні шаблони (наприклад, печатку конкретної організації), створи `presets.local.json` поряд:

```json
{
  "my-company": {
    "label": "Моя компанія",
    "shape": "circle", "size": "40", "color": "#1a4fa3",
    "topText": "ТОВ «Моя Компанія»", "topSize": 2.6,
    "bottomText": "ЄДРПОУ 12345678", "bottomSize": 2.0,
    "lines": [
      { "text": "ОСНОВНА", "size": 2.6, "weight": "bold" }
    ]
  }
}
```

`presets.local.json` ігнорується git-ом і ніколи не потрапить у публічну версію.

## Конвертація у криві

Поставити галочку **«Текст → криві»** в шапці. При експорті:
- кожен `<text>` замінюється на `<path>` через `opentype.js` (Roboto / Roboto Slab / Roboto Mono / Antonio / Anton — open-source шрифти)
- кожен `stroke` (кільце, рамка) перетворюється на заповнений region (annulus / frame)
- результат однаково виглядає в браузері, Inkscape, CorelDraw, Fusion 360

## DXF для Fusion 360 / AutoCAD

Кнопка **⬇ DXF** примусово конвертує текст у криві і виводить файл у форматі AutoCAD R12 ASCII (AC1009). Імпортується через `Insert → Insert DXF` у Fusion 360.

## Стек

- Чистий HTML / CSS / Vanilla JavaScript — без фреймворків і збірки
- [opentype.js](https://opentype.js.org/) — конвертація тексту у криві (CDN)
- Шрифти Roboto family + Antonio + Anton + Bebas Neue + Oswald — Apache / OFL ліцензії, в `fonts/`
