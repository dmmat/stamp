# Stamp Generator (Україна)

Простий статичний веб-застосунок для генерації відбитків печаток за українськими стандартами. Без серверу, без білду — відкриваєш `index.html` у браузері.

## Файли

- `index.html` — UI, всі контроли (форма, розмір, шрифт, тексти, лінії)
- `style.css` — стилі панелі та прев'ю
- `app.js` — генератор SVG (вся логіка тут, ванільний JS, без залежностей)

## Архітектура

- SVG рендериться в `mm` (`viewBox="0 0 W H"`, `width="Wmm"`) — експорт відразу в реальному розмірі
- Прев'ю масштабує SVG під 96 dpi (1 мм ≈ 3.78 px)
- `buildSvg()` — єдина точка побудови; повертає чистий SVG-вузол. Експорт SVG/PNG і прев'ю всі викликають її
- PNG експортується через canvas з масштабом 12× (~300 dpi)

## Стандартні розміри

- Кругла: 38, 40, 42, 45 мм (велика); 30, 25 мм (мала, нотаріус)
- Овальна: 50×35, 55×35, 60×40 мм
- Прямокутна: 60×25, 70×30, 50×20 мм
- Трикутна: 45, 50 мм

## Запуск

```bash
xdg-open index.html
# або підняти простий сервер:
python3 -m http.server 8000
```

## Принципи

- Жодних збірок, бандлерів, фреймворків — тільки HTML/CSS/JS
- Жодного мережевого трафіку: всі дані та генерація локально
- SVG як основний формат експорту, PNG — додатковий
